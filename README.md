# 🍰 Cake Catcher

> A web-based game project created to explore and master modern web game development by integrating **Vue.js**, **Vite**, and **Phaser 3**.

---

## 🎮 Game Overview
Cake Catcher is a "Catch & Avoid" style game where players must collect falling cakes while avoiding obstacles to achieve the highest score. This project serves as a practical study on handling game state within a reactive web framework.

## ✨ Key Features
* **Reactive UI:** Seamlessly integrated Vue.js components for menus and overlays.
* **Phaser Engine:** Smooth 2D gameplay mechanics using the Phaser 3 framework.
* **Database Integration:** High-score tracking and user data management via **Supabase**.
* **Optimized Performance:** Fast development and build process powered by **Vite**.

## 🛠️ Technologies Used
* **Frontend:** [Vue.js](https://vuejs.org/)
* **Bundler:** [Vite](https://vitejs.dev/)
* **Game Engine:** [Phaser 3](https://phaser.io/)
* **Backend/Database:** [Supabase](https://supabase.com/)

## 🚀 Getting Started

### Prerequisites
* [Node.js](https://nodejs.org/) (Version 16.x or higher recommended)
* npm

### Installation & Execution
1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/your-username/cake-catcher.git](https://github.com/your-username/cake-catcher.git)
    cd cake-catcher
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Run for development:**
    ```bash
    npm run dev
    ```
4.  **Build for production:**
    ```bash
    npm run build
    ```

## 📖 How to Play
* **Objective:** Catch as many cakes as possible to increase your score.
* **Controls:** Move: Use the Mouse to move the character left and right.
                Jump: Right-click to jump and reach higher cakes or dodge ground obstacles.
* **Avoid:** Stay away from 🍅 Tomatoes and 👾 Mimics! to keep the game going!
