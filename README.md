# Cake Catcher

Welcome to the **Cake Catcher** repository! 

This project was created as an educational endeavor to learn and practice modern web game development. The primary goal of this repository is to gain hands-on experience by integrating **Vue.js**, **Vite**, and the **Phaser 3** framework together.

## 🚀 Project Setup

If you would like to run this project locally, please follow the instructions below:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation & Execution

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/cake-catcher.git
   cd cake-catcher
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Compile and hot-reload for development**
   ```bash
   npm run dev
   ```

4. **Compile and minify for production**
   ```bash
   npm run build
   ```

## 🛠️ Technologies Used
- [Vue.js](https://vuejs.org/)
- [Vite](https://vitejs.dev/)
- [Phaser 3](https://phaser.io/)
- [Supabase](https://supabase.com/)
